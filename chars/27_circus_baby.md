# Circus Baby

- Spawns Bidybabs. There are 32 possible positions. The amount is equal to Circus Baby's AI level.
- Each camera has no more than 4 possible Bidybab locations.

Circus Baby is a "6AM jumpscare" animatronic and is checked before the 6AM cutscene shows up. This includes all boss fights, including Salvage. The game checks for each 6AM jumpscare animatronic, and if they would jumpscare you, the game deducts beartraps or jumpscares you for each one of them.
